# Expense Split App

Aplicación web desarrollada en Angular para dividir gastos entre personas
de forma simple y ordenada.

## 🚀 Descripción
El proyecto permite cargar gastos, participantes y calcular automáticamente
cuánto debe pagar cada persona y a quién.

Fue desarrollado como proyecto personal para practicar Angular,
componentes, servicios y manejo de estado.

## 🛠 Tecnologías
- Angular
- TypeScript
- HTML / CSS
